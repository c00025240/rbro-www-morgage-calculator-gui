Usage Guidelines

Small Spacing (0-16px)

Micro adjustments and tight layouts

--spacing-0 Resets, zero margins
--spacing-1 Icon gaps, fine adjustments
--spacing-4 Base unit, standard gaps

Medium Spacing (24-48px)

Component and section spacing

--spacing-5 Card padding, section gaps
--spacing-6 Layout sections, major spacing
--spacing-8 Page sections, hero spacing

Large Spacing (64-96px)

Page layout and major sections

--spacing-10 Page headers, major breaks
--spacing-12 Page sections, large breaks
--spacing-16 Page layout, enormous spacing

Implementation

CSS Custom Properties

.component {{ '{' }}
  padding: var(--spacing-4);
  margin-bottom: var(--spacing-6);

.component {{ '{' }}
  padding: $spacing-4;

{{ '}' }}
