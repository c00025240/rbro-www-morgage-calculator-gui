Official Usage Guidelines

Brand Colors (#fee600)

Use the official Raiffeisen yellow as the primary brand color for CTAs, highlights, and key UI elements. This is the core brand identifier.

Accent Colors (Teal)

Use the teal accent family for secondary actions, links, and complementary elements. These provide visual hierarchy while maintaining brand consistency.

Utilitarian Colors

Use semantic colors consistently: green for success (loan approval), red for errors (rejection), orange for warnings, and blue for information.

Accessibility & Contrast

All color combinations meet WCAG AA standards. Use the provided text colors with their corresponding background colors for optimal readability.
