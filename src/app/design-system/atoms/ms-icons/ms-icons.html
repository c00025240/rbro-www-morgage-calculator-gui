Usage Guidelines

Icon Sizes

  • Small (16px): Use in compact interfaces, badges, or inline with text
  • Medium (24px): Standard size for most UI elements
  • Large (32px): Use for featured actions or emphasis
  • X-Large (48px): Use for hero sections or major features

Icon Styles

  • Outlined: Default style, use for most interface elements
  • Filled: Use for active states, selected items, or emphasis
  • Half: Special style for ratings or progress indicators

Accessibility

  • Always provide alt text or aria-labels for icons
  • Use sufficient color contrast (4.5:1 minimum)
  • Don't rely solely on icons to convey meaning
  • Test with screen readers and keyboard navigation
