Implementation Guide

Best practices for implementing radius tokens in your projects

CSS Custom Properties

Modern, flexible implementation

/* Basic usage */
.component {{ '{' }}
  border-radius: var(--radius-4);
{{ '}' }}

/* Asymmetric radius */
.modal {{ '{' }}
  border-radius: var(--radius-12) var(--radius-12) 
                 var(--radius-4) var(--radius-4);
{{ '}' }}

/* Responsive radius */
.card {{ '{' }}
  border-radius: var(--radius-8);
{{ '}' }}

@media (max-width: 768px) {{ '{' }}
  .card {{ '{' }}
    border-radius: var(--radius-4);
  {{ '}' }}
{{ '}' }}

SCSS Variables

Enhanced with mixins and utilities

/* SCSS variables */
.component {{ '{' }}
  border-radius: $radius-4;
{{ '}' }}

/* Utility mixins */
@mixin radius-card() {{ '{' }}
  border-radius: $radius-8;
  overflow: hidden;
{{ '}' }}

/* Helper functions */
@function radius($size) {{ '{' }}
  @return map-get($radius-tokens, $size);
{{ '}' }}

.dynamic {{ '{' }}
  border-radius: radius(12);
{{ '}' }}
