Typography Guidelines

Hierarchy

Use consistent typography hierarchy to guide users through content. Display styles for major headers, headings for sections, and body text for content.

Responsive Design

Typography scales automatically across devices. Desktop uses larger sizes, tablet provides medium sizing, and mobile optimizes for readability.

Accessibility

All typography meets WCAG AA contrast requirements. Use appropriate font weights and sizes to ensure readability for all users.

Brand Consistency

Amalia font reflects Raiffeisen's brand identity. Use consistently across all mortgage simulator interfaces for brand recognition.

Implementation

CSS Custom Properties

/* Use typography tokens */
.my-heading {
  font-family: var(--font-primary);
  font-size: var(--text-h1-size);
  font-weight: var(--text-h1-weight);
  line-height: var(--text-h1-line-height);
  letter-spacing: var(--text-h1-spacing);
}
